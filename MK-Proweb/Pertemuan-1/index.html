Frontend vs Backend

Frontend

  • Antarmuka Pengguna (UI)
  • Pengalaman Pengguna (UX)
  • Logika Client-side
  • React.js, HTML, CSS

Backend

  • Logika Server
  • Manajemen Database
  • Pengembangan API
  • Laravel, PHP, MySQL

Client-Server Workflow

Browser (client) mengirim permintaan (request) ke server, lalu server mengirim respons. Proses ini dapat melibatkan pertukaran data JSON melalui RESTful API. Frontend dan Backend berkomunikasi menggunakan endpoint tertentu yang disepakati.

RESTful API Basics

  • GET: Membaca data
  • POST: Membuat data baru
  • PUT/PATCH: Memperbarui data
  • DELETE: Menghapus data

Contoh Request-Response

// Contoh Request dari Frontend
fetch('/api/pengguna')
    .then(response => response.json())
    .then(data => {
        console.log('Data Pengguna:', data);
    });
