Frontend vs Backend

Frontend

  • Antarmuka Pengguna (UI)
  • Pengalaman Pengguna (UX)
  • Logika Client-side
  • React.js, HTML, CSS

Backend

  • Logika Server
  • Manajemen Database
  • Pengembangan API
  • Laravel, PHP, MySQL

Teknologi Frontend

  • HTML5: Struktur Dokumen
  • CSS3: Styling & Layout
  • JavaScript: Interaktivitas
  • React: Component-Based UI

Teknologi Backend

  • PHP: Server-side Programming
  • Laravel: PHP Framework
  • MySQL: Database Management

Konsep Dasar RESTful API

  • GET: Membaca data
  • POST: Membuat data baru
  • PUT/PATCH: Memperbarui data
  • DELETE: Menghapus data

Contoh Implementasi API

// Contoh Penggunaan API di Frontend
const tambahPengguna = async (dataPengguna) => {
    const response = await fetch('/api/pengguna', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json'
        },
        body: JSON.stringify(dataPengguna)
    });
    return response.json();
};
