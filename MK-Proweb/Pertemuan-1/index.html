Frontend vs Backend

Frontend

  • Antarmuka Pengguna (UI)
  • Pengalaman Pengguna (UX)
  • Logika Client-side
  • React.js, HTML, CSS

Backend

  • Logika Server
  • Manajemen Database
  • Pengembangan API
  • Laravel, PHP, MySQL

Client-Server Workflow

Browser (client) mengirim permintaan (request) ke server, server mengirim respons. Proses ini dapat melibatkan pertukaran data JSON melalui RESTful API.

Langkah Instalasi

1. Install Node.js:

https://nodejs.org/

2. Install Git:

https://git-scm.com/

3. Install Laravel via Composer:

composer global require laravel/installer

RESTful API Basics

  • GET: Membaca data
  • POST: Membuat data baru
  • PUT/PATCH: Memperbarui data
  • DELETE: Menghapus data
