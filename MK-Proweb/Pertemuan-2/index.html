State Management

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

Laravel Basics (MVC)

Model

  • Representasi data
  • Interaksi dengan database
  • Logika bisnis

View

  • Tampilan untuk user
  • Template Blade
  • Response JSON untuk API

Controller

  • Mengatur alur aplikasi
  • Memproses request
  • Mengembalikan response

Routing di Laravel

Route::post('/users', [UserController::class, 'store']);
Route::put('/users/{id}', [UserController::class, 'update']);
Route::delete('/users/{id}', [UserController::class, 'destroy']);

Controller Example

class UserController extends Controller
{
    public function index()
    {
        $users = User::all();
        return response()->json($users);
    }

    public function store(Request $request)
    {
        $user = User::create($request->all());
        return response()->json($user, 201);
    }
}

Integrasi Frontend-Backend

Alur Kerja:

  • React Component membuat HTTP request ke Laravel endpoint
  • Laravel Controller memproses request
  • Controller mengembalikan response JSON
  • React Component memperbarui UI berdasarkan response
