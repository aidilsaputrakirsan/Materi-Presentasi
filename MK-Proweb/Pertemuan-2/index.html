Contoh Implementasi React.js

// Counter Component
function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => 
        setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

Contoh Implementasi Laravel

// UserController.php
public function index()
{
    $users = User::all();
    return response()->json($users);
}
// routes/api.php
Route::get('/users', [
    UserController::class, 
    'index'
]);

Integrasi Frontend-Backend

Alur Kerja:

  1. React Component membuat HTTP request ke Laravel endpoint
  2. Laravel Controller memproses request
  3. Controller mengembalikan response JSON
  4. React Component memperbarui UI berdasarkan response
